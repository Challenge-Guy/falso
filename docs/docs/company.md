# Company

### `randBrand`

Generate a random brand.

```ts
import { randBrand } from '@ngneat/falso';

randBrand();
randBrand({ length: 10 });
```

### `randCompanyName`

Generate a random company name.

```ts
import { randCompanyName } from '@ngneat/falso';

randCompanyName();
randCompanyName({ length: 10 });
```
